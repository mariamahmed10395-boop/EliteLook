Drop your product photos in this folder.

Each file name must match the "image" path used in
src/data/products.json, e.g.:

  src/data/products.json -> "image": "images/product1.jpg"
  actual file location    -> public/images/product1.jpg

Until a file exists, the site shows a soft placeholder
("Image coming soon") in its place automatically — nothing
else needs to change.
